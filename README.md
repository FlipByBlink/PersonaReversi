Description
------------
Reversi with Spatial Personas.

- Simple design.

- Support Spatial SharePlay.


Notice

- To use SharePlay, a FaceTime call is required.

- You must set up your persona on the OS beforehand.

- The persona appears as part of FaceTime/SharePlay.
